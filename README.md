# Python Projects

Small Python projects, one folder per project.

- [`book-price-scraper`](book-price-scraper) — web scraper that extracts book
  titles, prices and stock status from the
  [Books to Scrape](https://books.toscrape.com/) sandbox into a CSV file.

## Running a project

```bash
cd <project>
pip install -r requirements.txt
python scraper.py
```
