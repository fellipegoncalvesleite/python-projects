# Python Projects

Small Python projects, one folder per project.

- [`book-price-scraper`](book-price-scraper) — web scraper that extracts book
  titles, prices and stock status from the
  [Books to Scrape](https://books.toscrape.com/) sandbox into a CSV file.
- [`password-generator`](password-generator) — dependency-free command-line
  tool for generating cryptographically secure passwords.

## Running a project

Each project has its own README with setup and usage instructions.
